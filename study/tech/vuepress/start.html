# 如何搭建最简单的 vuepress 站点

# 依赖环境

  1. Node，如果你需要版本管理，可以通过nvm、nvm-windows引导安装node。这里有nvm教程、nvm-windows教程
  2. 如果你不喜欢npm，可以安装yarn，安装体验更好

# 创建项目

新建一个blog目录，按照下面的目录结构创建文件。

├─docs                  // 基础目录
| ├─.vuepress           // vuepress配置目录
| | └─config.js         // 基本配置
| └─README.md           // 主页内容
└─package.json          // 项目配置

提示

/开头表示当前目录下的绝对路径

/docs/.vuepress/config.js

module.exports = {
  title: '网站的标题，头部导航' // 如果不配置，无法返回主页
}

/package.json

{
  "scripts": {
    "docs:dev": "vuepress dev docs",  // 调试
    "docs:build": "vuepress build docs" // 构建
  },
  "devDependencies": {
    "vuepress": "^1.5.4"
  }
}

/README.md

Hello VuePress

创建之后，运行

npm install // yarn install
npm run docs:dev

浏览器访问http://localhost:8080就可以看到Hello VuePress
